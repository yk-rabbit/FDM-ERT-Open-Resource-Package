# FDM-ERT Open Resource Package

This repository contains the **raw experimental data**, **processing scripts**, and **figure reproduction guides** related to the research paper on Frequency Division Multiplexing Electrical Resistance Tomography (FDM-ERT).

---

## 📂 Repository Structure

- **`Raw Data, and Processing Scripts/`**: Contains original `.xlsx` data files (e.g., 1Hz, 2Hz, 4Hz) and the Matlab analysis scripts.
- **`Figure Reproduction Guide/`**: Contains detailed documentation (`.docx`) and reference data for reconstructing the inversion results and specific measurement modes.

---

## 📊 Data Processing & Visualization

### 1. Requirements
* **MATLAB**: Recommended version **R2019b** or later.
* **Microsoft Excel**: Required for reading raw data files.

### 2. How to Run
1. Ensure the `.m` scripts and the corresponding `.xlsx` data files are in the same directory.
2. Open and run `Plot_Figure_12.m` (or other scripts) in MATLAB.
3. Select the corresponding Excel file as prompted by the script to generate the resulting curves.

---

## 🖼️ Inversion Figure Reproduction
